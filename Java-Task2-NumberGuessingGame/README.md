## 📖 About The Project

A classic **Number Guessing Game**, reimagined with a modern **glassmorphism GUI** — frosted glass cards, gradient backgrounds, and smooth interactive buttons. The computer picks a random number, and the player has limited attempts to guess it correctly, with real-time feedback and score tracking.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎚️ **Difficulty Levels** | Easy (1–50), Medium (1–100), Hard (1–200) |
| 🎯 **Smart Feedback** | Instant "Too High" / "Too Low" / "Correct" responses |
| 🔢 **Attempt Tracker** | Live progress bar that changes color as attempts run low |
| 🏆 **Score System** | Earn more points for guessing in fewer attempts |
| 🔁 **Replay Option** | Jump straight into a new round with "Play Again" |
| 🧊 **Glassmorphism UI** | Frosted-glass card, gradient background, hover-glow buttons |

## 🚀 How To Run

```bash
# 1. Clone the repository
git clone https://github.com/asad594/OIBSIP.git

# 2. Navigate to the project folder
cd OIBSIP/Java-Task2-NumberGuessingGame/src

# 3. Compile the program
javac NumberGuessingGame.java

# 4. Run the program
java NumberGuessingGame
```

---

## 🎮 How To Play

1. Select a **difficulty level** from the dropdown
2. Click **Start New Game**
3. Enter your guess and click **Guess** (or hit `Enter`)
4. Follow the hints — *Too High* or *Too Low*
5. Guess correctly before attempts run out to score points!
6. Click **Play Again** to start a new round
