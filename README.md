## 📋 Submission Workflow

Every task in this repository follows the same structured process:

1. 📖 **Review** — Understand task requirements and feature checklist
2. 💻 **Build** — Develop the project following the specified tech stack
3. 📤 **Push** — Commit to this repository with proper folder naming
4. 🎥 **Demo** — Record a walkthrough video (name, track, task title)
5. 📱 **Share** — Post demo on LinkedIn with `#oasisinfobyte`
6. 🤝 **Peer Review** — Engage with fellow interns' submissions
7. ✅ **Submit** — Complete the official task submission form
